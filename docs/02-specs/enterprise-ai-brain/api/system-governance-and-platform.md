# 系统治理、RBAC 与平台配置 API 索引

> 该分册组已按业务模块继续拆细。主入口见 [../api.md](../api.md)，全量接口清单见 [catalog.md](catalog.md)。

## 分册

| 分册 | 覆盖范围 |
|------|----------|
| [system-rbac-and-settings.md](system-rbac-and-settings.md) | 系统设置、用户、角色、菜单、权限矩阵、业务大脑 |
| [product-config.md](product-config.md) | 产品、迭代版本、模块、Git 仓库、相关系统归属、版本分支、版本驾驶舱 |
| [platform-settings-and-model-gateway.md](platform-settings-and-model-gateway.md) | 相关系统、模型网关配置、连接测试、模型调用日志 |
| [assistant-workbench.md](assistant-workbench.md) | AI 助手聊天、引用候选、动作草案、草案模板、助手效果指标、会话历史 |

## 维护规则

- 系统设置、权限、角色或业务大脑变更更新 `system-rbac-and-settings.md`。
- 产品配置、版本状态、产品模块、Git 仓库或版本驾驶舱变更更新 `product-config.md`。
- 模型网关、相关系统或模型日志变更更新 `platform-settings-and-model-gateway.md`。
- AI 助手、动作草案、引用候选或助手指标变更更新 `assistant-workbench.md`。

# 常见问题

## 登录提示 DEFAULT_CREDENTIALS_DISABLED

该错误表示当前环境禁用了种子账号。请使用真实用户登录，或由管理员在本地测试环境显式启用测试账号。公网、生产或非本地环境不应使用 `admin@example.com` / `admin123` 这类种子凭据。

## 数字校验答错后无法登录

数字校验是一次性挑战。答错或过期后，页面会刷新题目并清空旧答案。请按新题目重新填写答案后再登录。

## 钉钉账号未绑定

如果提示 `DINGTALK_ACCOUNT_NOT_BOUND`，说明当前钉钉身份还没有绑定 AI Brain 用户。已有账号应先用账号密码登录，再到个人中心绑定钉钉。没有账号的用户需要管理员创建或审批。

## 钉钉绑定提示 EXTERNAL_IDENTITY_CONFLICT

该错误表示当前钉钉身份已经绑定到其他 AI Brain 用户。请联系管理员在用户管理或外部身份管理中确认绑定关系，必要时解除错误绑定后再重试。

## 页面提示 FORBIDDEN 或 Permission denied

这表示后端权限点拒绝当前操作。请确认：

- 当前角色是否具备该菜单和接口权限。
- 当前用户是否具备目标产品的数据范围。
- 当前操作是否属于写操作、审批或高风险动作。

viewer 角色通常只能只读查看，不允许执行写操作。

## 全链路打不开或没有数据

全链路依赖业务主体之间的上下文关系。独立巡检报告、未关联需求的版本或缺少上下游关系的数据，可能只展示空状态或禁用入口。请先确认主体是否关联了需求、任务、Bug、知识或审计记录。

## 测试邮件失败

测试邮件失败时优先检查：

- SMTP Host 是否正确。
- 端口和 SSL/TLS 是否匹配。
- SMTP 用户名是否是完整邮箱。
- SMTP 密码/授权码是否使用邮箱服务商要求的安全密码或客户端专用密码。
- 测试收件人是否可接收外部邮件。

阿里企业邮箱 SSL 发信常用 `smtp.qiye.aliyun.com` 和 `465` 端口，密码应填写安全密码或客户端专用密码。
